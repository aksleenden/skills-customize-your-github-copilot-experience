# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master string manipulation, file input/output operations, and text processing techniques. Build a practical text processing application that reads files, manipulates text data, and outputs formatted results.

## 📝 Tasks

### 🛠️ String Manipulation and Parsing

#### Description

Create functions that perform various string operations including transforming text case, extracting patterns, removing unwanted characters, and parsing structured text. Practice working with string methods and understanding how to efficiently manipulate text data.

#### Requirements

Completed program should:

- Convert text between different cases (uppercase, lowercase, title case)
- Split and join strings appropriately for data processing
- Extract specific information from formatted text
- Remove whitespace, punctuation, or other specified characters
- Validate and format text according to specifications


### 🛠️ File Input/Output Operations

#### Description

Implement reading from and writing to files with proper error handling. Work with different file modes, handle multiple lines of text, and ensure data is correctly persisted to disk.

#### Requirements

Completed program should:

- Read text files line by line
- Write processed data to new files
- Handle file not found and read/write errors gracefully
- Preserve or modify file formatting as needed
- Work with relative file paths


### 🛠️ Text Processing Application

#### Description

Build a complete text processing application that combines string manipulation and file I/O. Create a practical tool that takes input text, applies multiple transformations, and produces formatted output.

#### Requirements

Completed program should:

- Accept user input or read from a file
- Apply multiple text processing operations sequentially
- Generate properly formatted output
- Provide feedback on processing results
- Save results to an output file
